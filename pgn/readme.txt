Code in this directory was copied from this (https://github.com/teodoran/pgn.net) repository.

The changes were made to files 'pgn.Parse/BasicChessParsers.fs' and 'pgn.Data/Basics/File.cs' to suit the bigger board of hexagonal chess. The file 'pgn.Data/MoveText/GameEndEntry.cs' was modified to resolve an issue with displaying non-standard symbols. Unneeded debug code was removed from files 'pgn.Parse/Bootstrap.fs' and 'pgn.Parse/MoveParsers.fs'.
